# AutoCONTIN
CONTIN wrapper, full automatic with batch mode.
Especially for light scattering systems from ***Brookhaven Inc.***

## File list
- AutoCONTIN_cmd.py
- AutoCONTIN.py
- CONTINWrapper.py
- contin-windows.exe

## Before Use
please put CONTIN executable file contin-windows.exe in folder C:\Program Files (x86)\CONTIN\
